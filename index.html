CSV File Requirements

Upload a CSV file containing user data to import, delete, or modify users in your PingOne environment.

File Requirements:

  • File must have a .csv extension
  • First row should contain column headers
  • Each subsequent row represents one user
  • Required columns: username, email (or other user attributes)
  • Use commas to separate values

The app will process each row and perform the selected action (Import/Delete/Modify) on your PingOne environment.

Troubleshooting Steps:
